Deskripsi: Project ini merupakan Sistem Manajemen Penginapan dengan studi kasus pengelolaan cabin pada penginapan "Lembayung Nirwana". 
Pada tugas akhir ini dilakukan pengembangan fitur autentikasi pengguna serta peningkatan pengalaman pengguna.

Fitur yang diperbarui
1. Login
Sistem menyediakan halaman login untuk autentikasi pengguna menggunakan username dan password.

2. Logout
Pengguna dapat keluar dari sistem, sehingga session akan dihapus dan akses kembali menjadi terbatas.

3. Remember Me (Cookies)
Jika fitur "Remember Me" dicentang saat login:
- Username akan disimpan menggunakan cookies
- Saat halaman login dibuka kembali, field username akan otomatis terisi

Jika tidak dicentang:
- Cookies akan dihapus dan username tidak disimpan

8. Validasi Login
Jika login gagal, menampilkan pesan error menggunakan komponen Bootstrap (alert)
