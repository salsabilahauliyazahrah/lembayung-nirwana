## Deskripsi
Pengembangan sistem manajemen penginapan berbasis Laravel dengan penambahan fitur autentikasi menggunakan Google OAuth melalui Laravel Socialite, integrasi Google reCAPTCHA pada halaman registrasi, serta pengujian fitur autentikasi menggunakan Laravel Testing.

## Fitur
- Login & Register
- Login dengan Google
- Google reCAPTCHA pada Register
- Role Admin dan User
- Dashboard Pengguna
- CRUD Cabin
- Upload Gambar Cabin
- Profile Management
- Relasi Product, Category, dan Brand
- Feature Testing Laravel
